Hash Functions & MACs

Hash Functions

  • Pre-image resistance: Given h, infeasible to find m where H(m)=h.
  • Second pre-image resistance: Given m1, infeasible to find m2 with H(m1)=H(m2).
  • Collision resistance: Infeasible to find any m1â‰ m2 with same hash.

Message Authentication Codes (MACs)

  • Require shared secret key.
  • Provide integrity and authenticity.
  • Common: HMAC using SHA-256.

Digital Signatures vs MACs

MACs use shared key (symmetric). Signatures use public/private key pairs (asymmetric).

Practice Questions

  1. What property of a hash function makes it infeasible to find the original message from the hash?
    Pre-image resistance.
  2. Which property prevents finding a second message that hashes to the same value as a given message?
    Second pre-image resistance.
  3. What is the main purpose of a Message Authentication Code (MAC)?
    To provide both integrity and authenticity of a message using a shared secret key.
  4. How does a digital signature differ from a MAC?
    Digital signatures use asymmetric keys (public/private) while MACs use a shared symmetric key.
